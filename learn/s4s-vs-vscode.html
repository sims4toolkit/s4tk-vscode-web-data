In a few ways, VS Code is similar to Sims 4 Studio. The file explorer is on the left, the editor is on the right, and their editors are both powered by Monaco (which means they use many of the same keyboard shortcuts, such as âŒ˜ CMDCTRL F to find / replace).

The biggest difference between them is that Sims 4 Studio works with packages while VS Code works with individual files on your computer. Most professional software development is done with individual files for a few reasons:

  • You have full control over your project's file structure
  • You're not restricted to programs that can read packages
  • You won't break your entire project if one file is corrupt
  • You can edit one file without reading/writing the entire package
  • You can use Git for source control and back your code up on GitHub
