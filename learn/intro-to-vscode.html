Getting Started

Visual Studio Code is a free, cross-platform code editor created by Microsoft. It is highly customizable thanks to its extension marketplace, where you can download add-ons that alter its appearance and/or abilities (just like mods!).

Download for Windows, macOS, or Linux at the above link.

VS Code works with actual files and folders on your computer's file system, rather than virtual documents within a package, like Sims 4 Studio does. Each project you work on in VS Code must exist within a single folder; when you open this folder in VS Code (File > Open Folder... in the menu bar), it is called a workspace.

Within your workspace, you can create, view, edit, move, rename, and delete files, just as you would in macOS FinderWindows Explorer, but with a rich code editor. The workspace folder can also contain special files, such as the S4TK Config or the .vscode folder, which tell VS Code how to behave while working on your current project.

You'll read more about the S4TK Config and .vscode folder in other articles.
