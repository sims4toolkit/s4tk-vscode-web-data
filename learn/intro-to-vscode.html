Getting Started

Visual Studio Code is a free, cross-platform code editor created by Microsoft. It is highly customizable thanks to its extension marketplace, where you can download add-ons that alter its appearance and/or abilities (just like mods!).

Download for Windows, macOS, or Linux at the above link.

VS Code works with actual files and folders on your computer's file system, rather than virtual documents within a package, like Sims 4 Studio does. Each project you work on in VS Code must exist within a single folder; when you open this folder in VS Code (File > Open Folder... in the menu bar), it is called a workspace.

Within your workspace, you can create, view, edit, move, rename, and delete files, just as you would in macOS FinderWindows Explorer, but with a rich code editor. The workspace folder can also contain special files, such as the S4TK Config or the .vscode folder, which tell VS Code how to behave while working on your current project.

You'll read more about the S4TK Config and .vscode folder in other articles.

User Interface

This section introduces areas of the UI that are relevant while using the S4TK extension. For a more complete introduction to the VS Code user interface in general, please read the official "User Interface" docs.

The majority of the time you're working in VS Code, you'll be looking at something like this:

Screenshot of VS Code with the S4TK Extension Active

  • File Explorer: View and edit the contents of your workspace folder. Click files to open them in a new editor tab.

    • Tip: Single-clicking will open a new temporary tab. To keep a file open, double-click it.

  • Editor Tabs: Switch between the editors you currently have open.

    • Tip: If a tab's title is written in italics, it is temporary, and will be replaced when you open another file.

  • Editor: View and edit the contents of a file.

    • Tip: You can have multiple editors open side-by-side for easy comparison.

  • Context Menu: Perform actions on a file/folder by right-clicking them in the explorer or editor tabs.

    • Tip: Many of S4TK's commands are within context menus, so you don't have to run them manually.

  • Information Message: Displays information such as confirmations, warnings, and errors.

    • Tip: S4TK makes use of these popups often, but you can silence some using the S4TK Config.
