S4TK VS Code v0.2.1 updates dependencies on the @s4tk library, which addresses some bugs. Despite its small scope, this is a critical update. Without the SimData fix, some of your SimData files may not work properly.

Bug Fixes

  • Fixed issue where formatting XML would sometimes erraneously put nodes on same line.
  • Fixed issue where certain SimDatas with adjacent booleans would serialize incorrectly.
