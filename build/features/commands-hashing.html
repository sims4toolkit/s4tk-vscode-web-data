Command Palette

To open the command palette, press ⌘ CMDCTRL ⇧ SHIFT P. Then, you can start typing one of the following commands:

Hash specific text

  • Hash text with FNV32
  • Hash text with FNV64
  • Hash text with FNV (custom bits)

Generate random hash

  • Generate random FNV32 hash
  • Generate random FNV64 hash
  • Generate random FNV hash (custom bits)

Random hashes are generated with a UUID and the current epoch stamp in milliseconds. The chance of a collision is exceedingly rare.

All six commands will show a popup like the one below; click any of its buttons to copy it to your clipboard. From left to right, the buttons will copy the hash as decimal, hex with a 0x prefix, and hex without a 0x prefix.
