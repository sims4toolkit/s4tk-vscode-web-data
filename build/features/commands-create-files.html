Command Palette

To open the command palette, press ⌘ CMDCTRL ⇧ SHIFT P. Then, you can run commands to create files:

  • Create S4TK Config: Creates a default s4tk.config.json file at your workspace root.
  • Create Sample S4TK Project: Generates a sample S4TK project setup (read more).
  • Create String Table (Binary): Prompts you for a name and location, and creates a binary STBL.
  • Create String Table (JSON): Prompts you for a name and location, and creates a STBL JSON.

These commands are non-destructive, meaning they will not overwrite files that already exist. Attempting to create a STBL that exists will open the existing file, while attempting to create a config or sample project will fail.

Context Menus

Right-click any folder within your workspace to bring up a context menu, where you'll see the following commands:

  • Create S4TK Config: Creates a default s4tk.config.json file in the folder.

    • Only visible if there is no config already loaded. For more info, read S4TK Config.

  • Create String Table (JSON): Prompts you for a name, and saves the STBL JSON to the folder.

Note that the sample workspace and binary STBL commands are intentionally excluded from the context menus, so as to reduce clutter, since they should rarely be used.
