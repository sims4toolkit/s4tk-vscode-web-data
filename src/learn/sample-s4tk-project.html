The best way to learn the S4TK VS Code extension is by jumping right on in and playing around with it. For this, there is a sample project baked-in to the extension, and you can generate it with a simple command.

Creating the Sample Project

Once you've installed the Sims 4 Toolkit extension in VS Code, open a new folder to use as your sample project's workspace (File > Open Folder... in the menu bar). Then, open the command palette with ⌘ CMDCTRL ⇧ SHIFT P, and type in "Create Sample S4TK Project" in the textbox that appears. Select it using ↓ / ↑, and press RETURNENTER.

The "Create Sample S4TK Project" Command

After generating the sample project, a file called HowToUseS4TK.md should automatically be opened in an editor. You can read through this for an explanation about the workspace that was just generated, otherwise, just poke around the files yourself, read their comments, and maybe run a build by right-clicking the s4tk.config.json file.
