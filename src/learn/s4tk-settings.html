The S4TK config only affects project-level settings, but there are some more options you can choose that affect how the S4TK extension behaves overall. This article explains how to view and edit settings, not which settings are available and what they mean. For those details, read the S4TK Settings feature page.

How to Access Settings

There are two ways to edit your VS Code settings, either with the settings menu or by editing the JSON file.

Using the settings menu

To open the settings menu, you can use âŒ˜ CMDCTRL ,. This will open a tab where you can view all of your VS Code settings, including settings for extensions. Click "Extensions" in the left-side menu, and scroll down to "Sims 4 Toolkit". From here, you can read about available settings and choose what you want them to be.

Sims 4 Toolkit Settings Within VS Code

Using the settings JSON

If you prefer to edit your settings manually, you can do that as well. To open the JSON file where your settings are stored on disk, press the "Open Settings (JSON)" button in the top-right of your settings UI tab.

VS Code "Open Settings (JSON)" Button

After clicking this, the following JSON file will appear. Of course yours will be different depending on your settings, but you can edit your S4TK settings by typing them in as a new property. This JSON file has full intellisense, and will also tell you when you've set something to an incompatible value.
