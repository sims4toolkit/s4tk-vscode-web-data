As v1.0.0 is the intial release of the S4TK VS Code extension, this page is just a list of its features at time of writing, rather than a traditional changelog that highlights new/fixed/changed/removed behavior. It also does not include details about individual features or information about how to use them - for that, please check the Features and Learn pages.

Initial Features

CodeLenses

  • S4TK Config

    • Available in: s4tk.config.json

    • Build
    • Dry Run
    • Release
    • New Package
  • String Table JSON

    • Available in: *.stbl.json

    • Convert to Array
    • Convert to Object
    • Copy as XML
    • Insert Metadata
    • New String
  • XML Tuning / SimData

    • Available in: *.xml

    • Format
    • Type
    • Group
    • Instance

Commands (Command Palette)

  • FNV Hashing
    • Generate random FNV32 hash
    • Generate random FNV64 hash
    • Generate random FNV hash (custom bits)
    • Hash text with FNV32
    • Hash text with FNV64
    • Hash text with FNV (custom bits)
  • S4TK Workspace
    • Add New String
    • Build S4TK Project
    • Build S4TK Project (Dry Run)
    • Build S4TK Project (Release)
    • Create Sample S4TK Project
    • Create S4TK Config
    • Reload S4TK Config
  • TS4 Files
    • Create String Table (Binary)
    • Create String Table (JSON)

Commands (Explorer / Tabs)

  • File Creation

    • Available on any folders

    • Create S4TK Config
    • Create String Table (JSON)
  • String Tables

    • Available on: *.stbl and *.stbl.json

    • Add New String
    • Set STBL as Default
  • S4TK Config

    • Available on: s4tk.config.json

    • Build S4TK Project
    • Build S4TK Project (Release)
  • XML Tuning

    • Available on: *.xml, excludes .SimData.xml

    • Copy as XML Reference

Custom Editors

  • Binary String Table Editor (view-only)

    • Renders: *.stbl

    • Button to refresh (for when file changes on disk)
    • Button to convert to JSON
  • Package Editor (view-only)

    • Renders: *.package

    • Button to refresh (for when file changes on disk)

JSON Schemas

  • Build Summary

    • Applies to: BuildSummary.json

  • String Table JSON

    • Applies to: *.stbl.json

  • S4TK Config

    • Applies to: s4tk.config.json

Settings

  • Default String Table Json Type

    • JSON name: s4tk.defaultStringTableJsonType

    Default value: "object"

  • Default String Table Locale

    • JSON name: s4tk.defaultStringTableLocale

    Default value: "English"

  • New Strings To Start Of String Table

    • JSON name: s4tk.newStringsToStartOfStringTable

    Default value: true

  • Show Config Loaded Message

    • JSON name: s4tk.showConfigLoadedMessage

    Default value: true

  • Show Config Unloaded Message

    • JSON name: s4tk.showConfigUnloadedMessage

    Default value: true

  • Show Copy Confirm Message

    • JSON name: s4tk.showCopyConfirmMessage

    Default value: true
