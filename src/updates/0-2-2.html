S4TK VS Code v0.2.2 is a minor update that includes the latest version of @s4tk/models and a new command.

New Tuning Type

Tunings with i="business_rule" will now automatically use the BusinessRule (B8E58C6C) tuning type.

New Command

"Rename / Rehash Tuning File"

Available in context menus for XML files. Renames a tuning and its paired SimData, if it has one, and automatically generates a new hash. If the file is known to require 31 or 32 bit hashes, the hash will be the correct size.

This command is nearly identical to the existing "Clone With New Name" command, but without keeping the original file.
