These are commands that relate to your S4TK project setup, rather than those for building.

Command Palette

To open the command palette, press ⌘ CMDCTRL ⇧ SHIFT P. Then, you can run commands for reloading the config, creating a config, or generating a sample project:

Workspace Commands in Command Palette

Reloading the config is generally not necessary because the config will automatically reload every time you save an edit, however, it can be used when the config is edited by an external program, such as TextEditNotepad.

Note that the "create" commands are non-destructive, meaning that they will never overwrite files that already exist. In fact, an error message will display if you attempt to run them in a workspace that already has an S4TK config.

Context Menus

You can generate the config or sample workspace by right-clicking the root workspace folder and selecting the commands in the context menu. For screenshots and more information, visit S4TK Config and Sample S4TK Project.
