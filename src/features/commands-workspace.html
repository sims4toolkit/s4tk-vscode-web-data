Command Palette

To open the command palette, press ⌘ CMDCTRL ⇧ SHIFT P. Then, you can run these commands:

  • Convert Folder to S4TK Project: Select a folder containing packages and/or loose TS4 resources, then select a folder to output the project to, and these files will be converted to a format that is compatible with the extension. The folder structure will be broken down by file type.

    • The S4TK project is guaranteed to use the same resource keys for every single file in the selected source folder. If inference doesn't match, an override comment is inserted for you.

  • Refresh Project Index: Refresh the index for this project. Probably never has to be used, but available just in case.

    • If you ever find that the index is incorrect or has fallen out of sync, please report an issue.
