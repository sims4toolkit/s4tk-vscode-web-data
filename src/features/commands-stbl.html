Command Palette

To open the command palette, press ⌘ CMDCTRL ⇧ SHIFT P. Then, you can use the command:

  • Add New String: Prompts you for a string to add to the default STBL.

    • Your default STBL is set in your s4tk.config.json. The key will be a cryptographically safe FNV32 hash (more info).

Context Menus

Right-click a string table (binary or JSON) either in the file explorer or editor tabs in order to access the following commands:

  • Add New String: Adds a string with a random hash to this STBL.

    • The key will be a cryptographically safe FNV32 hash (more info).

  • Set STBL as Default: Updates your config to set this STBL as the default.

The following commands are available on JSON STBLs only:

  • Create Fragment for STBL: Creates a STBL with the same meta data as this one, but marked as a fragment.
