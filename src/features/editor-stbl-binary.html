Binary string tables are not directly editable; they are view-only. It is recommended to use String Table JSONs in S4TK.

Viewer

The binary string table viewer displays all keys/value pairs in any *.stbl or 220557DA*.binary files, and supports using âŒ˜ CMDCTRL F to search. It does not, however, have any ability to add, edit, remove, or copy existing strings.

Commands

In addition to the String Table Commands, binary STBLs have two more at the top of the editor:

  • Reload: Refreshes the contents of the editor.

    • For use when the STBL changes, such as after using the "Add New String" command.

  • Convert to JSON: Creates a JSON string table with the same name and contents as this binary one.

    • For use when the STBL changes on disk, such as after using the "Add New String" command.
