The S4TK extension adds functionality on XML files for formatting, resource keys, and copying references.

CodeLenses

Tuning and SimData both contain the same CodeLenses at the top of the file:

Available CodeLenses in Tuning and SimData

You can format your XML by clicking "Format". It will use the number of spaces set in your VS Code settings.

Additionally, S4TK automatically infers resource keys for tuning and SimData, and will display them in the three other CodeLenses as hex numbers. If you need to override one and set it manually, then just click the CodeLens and a comment will be inserted. For more information, read Resource Keys.

If one of the resource key values says "Unknown", S4TK could not infer it, and it must be manually set.

Commands

There is only one command available on XML files, and it applies to Tuning only, not SimData. Right-click on the tuning file in either the file explorer or editor tabs, and you will see the following option:

"Copy as XML Reference" Command in the Context Menu

This will copy the tuning file as an XML reference, such as 12345<!--filename-->, so you can paste it into another tuning file.
