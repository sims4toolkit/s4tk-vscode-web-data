Hashing Commands in the Command Palette

"Hash text with" commands will prompt you for text to hash, while "Generate random" commands will create a cryptographically-safe randomized hash*. All 6 of these commands will result in a popup similar to the one below, and you can press any of its buttons to copy it to your clipboard.

* Hashes are generated with a random UUID and the current epoch time in milliseconds. The chance of a collision is exceedingly rare.

Message Popup for Copying Hashes

From left to right, the buttons will copy the hash as decimal, hex with a 0x prefix, and hex without a 0x prefix.
