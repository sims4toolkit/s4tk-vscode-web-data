You can create string tables, configs, and sample projects quickly with commands.

Command Palette

To open the command palette, press ⌘ CMDCTRL ⇧ SHIFT P. Then, you can run commands to create files:

Workspace Commands in Command Palette

Creating a STBL from the command palette will prompt you for a directory to create it in. The config and sample project will always generate at the root of the current VS Code workspace, since that is where the config must be.

These commands are non-destructive, meaning they will not overwrite files that already exist. Attempting to create a STBL that exists will open the existing file, while attempting to create a config or sample project will fail.

Context Menus

You can generate the config or sample workspace by right-clicking the root workspace folder and selecting the commands in the context menu. For screenshots and more information, visit S4TK Config and Sample S4TK Project.

You can also create a JSON string table with a context menu command, but only when there is an S4TK config file loaded. Right-click on any folder within your workspace in order to create a string table JSON.

Context Menu Showing Option to Create STBL JSON

The command to create a binary string table is intentionally excluded, as it's not recommended to use them in the first place.
